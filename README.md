# Bus Ticket Booking – Oracle APEX Application

## Overview
This is an Oracle APEX-based Bus Ticket Booking application that allows
users to manage customers and book bus tickets.

## Features
- User authentication and role management
- Bus ticket booking
- Booking details management

## Deployment (Split Files)

### Application Import
1. Zip the contents of the `apex` folder.
2. Login to Oracle APEX.
3. Go to App Builder → Import.
4. Upload the zip file.
5. Install supporting objects when prompted.
6. Run the application.

### Database Objects
All required tables and PL/SQL objects are created via:
- `apex/supporting_objects/installation/install.sql`

(An additional DDL backup is available in `database/`.)

## Tech Stack
- Oracle APEX
- Oracle Database
- PL/SQL
